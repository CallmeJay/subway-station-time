## 隐私协议 (合规必填)

最简化版本 (上线前必填):

```
本小程序「下一班 · 地铁助手」(以下简称"我们") 致力于保护用户隐私.

1. 不收集信息
   本小程序不主动收集任何用户个人信息 (姓名/手机/位置等).

2. 本地存储
   用户配置 (收藏、默认开屏站、主题偏好等) 仅保存在用户设备本机
   (通过 wx.setStorage), 不上传任何服务器.

3. 网络通信
   仅与微信云开发数据库通信, 用于拉取地铁时刻表数据 (公开数据).
   不传输任何用户身份信息.

4. 第三方
   除微信云开发外, 不集成任何第三方 SDK.

5. 头像与昵称
   通过微信原生 chooseAvatar / type=nickname 控件获取, 仅本地保存,
   不上传服务器.

如有疑问, 请通过小程序「我的」→「反馈与建议」联系开发者.
```

## 服务声明 (类目要求)

工具类基本无额外要求. 如选「出行 → 公交地铁」类:
- 个人主体: 多数情况无法使用此类目
- 企业主体: 需 ICP 备案 + 数据合作授权证明

**推荐**: 选「工具 → 实用工具」, 个人主体可过审.

## 图标设计建议

- **主元素**: 「下」字 / 地铁列车 / 时钟 / "→" 箭头
- **配色**: 主色 #A4343A (1 号线红, DB11 国标), 辅助 #FFFFFF
- **风格**: 极简圆角矩形或圆形, 与 v4 卡片视觉一致
- **尺寸**: 144×144 (上传必须), 备用 200×200 / 1024×1024

简单可行的占位方案:
1. 用任何 emoji 字体把「下」或「→」放圆形红底
2. 微信开发者工具 → 详情 → 项目设置 → 上传 logo
